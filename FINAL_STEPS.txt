ÉTAPE 1 : Configurer les Secrets Supabase (2 min)
──────────────────────────────────────────────────

1. Ouvrir ce lien :
   👉 https://supabase.com/dashboard/project/nykctocknzbstdqnfkun/settings/functions

2. Scroller en bas jusqu'à "Edge Function Secrets"

3. Cliquer sur "Add new secret" pour chaque :

   Nom : GITHUB_TOKEN
   Valeur : ghp_votre_token_github
   (obtenir sur : https://github.com/settings/tokens)

   Nom : TELEGRAM_BOT_TOKEN
   Valeur : votre_bot_token_telegram

   Nom : TELEGRAM_CHAT_ID
   Valeur : votre_chat_id

   Nom : TELEGRAM_NOTIFICATIONS
   Valeur : true

4. Cliquer sur "Save"

✅ Les secrets seront utilisés par l'Edge Function !

──────────────────────────────────────────────────

ÉTAPE 2 : Déployer Dashboard sur Vercel (2 min)
──────────────────────────────────────────────────

1. Ouvrir :
   👉 https://vercel.com

2. Cliquer sur "Add New..." → "Project"

3. Cliquer sur "Import Git Repository"

4. Sélectionner : DafnckStudio/github-security-scraper

5. Configuration :
   Framework Preset : Other
   Root Directory : ui
   Build Command : (laisser vide)
   Output Directory : .
   Install Command : (laisser vide)

6. Cliquer sur "Deploy"

7. Attendre 1-2 minutes

✅ Dashboard sera accessible à :
   https://github-security-scraper.vercel.app
